## 功能简介

所有公共组件的合集，建议平时开发直接使用CommonWrapper的最新版

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:common-wrapper:+'
```

## 组件功能

主要是二次封装非 CommonTest 以外的所有公共组件
