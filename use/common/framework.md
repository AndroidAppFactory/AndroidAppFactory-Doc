## 功能简介

通用应用开发的公共框架，包含常量定义、调试模式、路由管理、通用页面，结合场景的通用弹框，基础资源以及对于所有基础组件的必要初始化等

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:framework:+'
```
