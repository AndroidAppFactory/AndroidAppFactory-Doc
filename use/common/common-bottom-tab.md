## 功能简介

基于底部菜单的通用主页及不同的底部菜单样式

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:common-bottom-tab:+'
```

## 组件功能

### CommonMainFragment

- 基于底部菜单的通用主页

### SimpleBottomBarTab

- 支持通过颜色控制不同选中态的TAB

### SvgaBottomBarTab

- 支持SVGA，以及默认和选中使用不同图标的TAB
