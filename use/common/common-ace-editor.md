## 功能简介

基于AceEditor实现的文本编辑查看工具

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:common-ace-editor:+'
```

## 组件功能

### AceEditActivity

基于AceEditor实现的文本编辑查看工具，目前支持查看文本、文本自动换行、分享文本

### AceEditFragment

基于AceEditor实现的文本编辑查看工具，方便内嵌到项目其余位置
