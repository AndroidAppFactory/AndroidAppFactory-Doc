## 功能简介

基于公共框架的使用系统原生进一步封装的拍照、相册选择、裁剪图片

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:common-photos:+'
```

## 组件功能

### Activity的扩展

- 获取无需相册权限的拍照、裁剪目录及图片

- 无需相册权限的拍照

- 基于 [PermissionManager](./../libs/ui/lib-permission.md#permissionmanager) 完成权限设置及检查的相册照片选择
