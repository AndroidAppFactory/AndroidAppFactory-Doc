## 功能简介

基于公共框架的图片裁剪方案，支持图片按比例裁剪、缩放、旋转等

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:common-crop:+'
```

## 组件功能

### CropUtils

- 拉起图片裁剪页面，通过Intent 传递参数，结果通过 onActivityResult 回调
