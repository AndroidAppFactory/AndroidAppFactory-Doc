## 功能简介

基于公共框架的图片处理扩展，例如高斯模糊等

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:common-image:+'
```

## 组件功能

### BlurTransformation

Glide 支持高斯模糊的转换方式


### HeadIconBuildFactory

基于 HeadIconBuilder 的，支持使用缓存的扩展类
