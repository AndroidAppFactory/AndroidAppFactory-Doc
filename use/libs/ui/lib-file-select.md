## 功能简介

提供了一个文件选择器，选择结果以onActivityResult的结果返回

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-file-select:+'
```

### FileSelectTools
