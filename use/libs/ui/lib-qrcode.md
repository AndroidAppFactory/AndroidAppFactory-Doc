## 功能简介

一款基于zxing的二维码识别类库，目前已经被二次封装

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-qrcode:+'
```

## 组件功能

二维码扫描、识别、生成的工具类，目前仅合入进来，需要进一步整理
