## 功能简介

View的点击区扩大，最大可以扩大到父布局的大小

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-touchregion:+'
```

## 组件功能

### ViewExtForTouchRegion

- 基于Kotlin的扩展函数添加的 View 点击区扩展，可以指定大小，最大可以扩大到父布局的大小
