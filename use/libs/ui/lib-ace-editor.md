## 功能简介

基于 AceEditor 实现的文本编辑查看工具

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-ace-editor:+'
```

## 组件功能

### AceEditorView

- 对 AceWebView 的封装

### AceWebView

- 基于 Ace 定制的文本、代码编辑查看工具的View
