## 功能简介

资源反射、基础的UI通用ICON

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-ui-common:+'
```

## 组件功能

### Res

- 各种资源的反射获取，包括string、drawable、mipmap、layout、id、style、color等


### ColorTools

- Color的特殊操作，给颜色添加透明度，计算颜色的属性（亮色、暗色）
