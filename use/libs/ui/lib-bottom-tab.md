## 功能简介

通用的自定义UI，目前有底部按钮栏，带红点的角标TextView

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-bottom-tab:+'
```
