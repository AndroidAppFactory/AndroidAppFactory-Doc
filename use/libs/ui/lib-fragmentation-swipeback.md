## 功能简介

单Activity ＋ 多Fragment 模式的Fragment管理框架。对应GitHub：https://github.com/YoKeyword/Fragmentation 目前已经做了二次封装

## 组件信息

### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-fragmentation-swipeback:+'
```
