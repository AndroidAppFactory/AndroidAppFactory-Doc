## 功能简介

对ViewPager的扩展，目前仅增加不支持滑动切换页面的ViewPager

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-view-pager:+'
```

## 组件功能

### NoScrollViewPager

- 不支持滑动切换页面的ViewPager
