## 功能简介

配置管理相关，支持读取本地配置文件、支持配置保存本地

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-config:+'
```

## 组件功能

### Config

- 配置加载、读取、写入等

- 支持基本数据类型、整个Map，整个 Json
