## 功能简介

图片裁剪的native 实现

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-crop:+'
```


## 组件功能

### CropAction

- 根据传入的图片路径，大小，缩放比例，旋转角度等裁剪图片
