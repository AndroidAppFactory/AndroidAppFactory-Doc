## 功能简介

文字转语音组件的封装

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-tts:+'
```

## 组件功能

### LibTTS

- 文字转语音封装，需要系统内置引擎，

- 支持设置语速和语调、支持各种形式的语音播报
