## 功能简介

简单封装的日志工具类，支持注册自定义日志实现

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-log:+'
```

## 组件功能

### ZLog

- 日志工具类的各类日志打印方式
    
###  LogImplForLogcat

- 使用系统Logcat的日志实现
    
### LogImpl

- 自定义日志实现的抽象
