## 功能简介

提供ZIP 文件的压缩和解压缩，支持2G以上超大文件

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-zip:+'
```

## 组件功能

### ZipUtils

- ZIP文件压缩与解压缩

- 支持超过2G大文件

- 支持不解压查看列表，解压部分文件
