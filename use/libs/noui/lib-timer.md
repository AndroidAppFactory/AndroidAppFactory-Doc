## 功能简介

基于Timer封装的定时器

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-timer:+'
```

## 组件功能

### TaskManager

- 定时任务的添加、删除

### BaseTask

- 定时任务的基本信息，例如运行周期、任务内容等

### TimerProcessManager

- 基于倒计时的进度管理
