## 功能简介

通用路由的annotation工具组件，用于自动生成路由表

## 组件信息

### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-router-annotation:+'
```
