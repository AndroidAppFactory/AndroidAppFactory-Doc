## 功能简介

Widget 及锁屏相关服务的合集，建议平时开发直接使用LibScreenWrapper的最新版

## 组件信息

#### 引用仓库

引用仓库可以参考 [组件使用](./../start.md) 中添加依赖的部分

#### 组件使用

```groovy
implementation 'com.bihe0832.android:lib-wrapper-screen:+'
```

## 组件功能

主要是二次封装所有 Widget 及锁屏相关服务，方便日常调用
